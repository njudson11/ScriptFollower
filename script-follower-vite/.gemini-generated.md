# Script Follower

This project is a Vue.js web application built with Vite that also has the capability to be packaged as a desktop application using Electron.

## Overview

The "Script Follower" application is designed to assist users in following a script. It provides several key features:

- **Document Viewing:** Users can load and view script documents in various formats (including .docx, .odt, and .pdf).
- **Speech Recognition:** The application uses speech recognition to automatically detect the user's place in the script and can autoscroll to the current line.
- **Sound Playback:** The application can play sound cues that are embedded in the script.
- **PWA Support:** The application is a Progressive Web App, allowing it to be "installed" on devices for a more native-like experience.

## Tech Stack

- **Framework:** Vue.js 3 (with Composition API)
- **Build Tool:** Vite
- **Desktop Framework:** Electron
- **Key Libraries:**
    - `@fortawesome/vue-fontawesome` for icons.
    - `@xmldom/xmldom`, `jszip`, `mammoth`, `pdfjs-dist` for document processing.
    - `vite-plugin-pwa` for Progressive Web App features.
